Deck Toolbox

Quick-fix scripts for your Steam Deck.

Made by EERIE | Buy Me a Coffee
Generate .cue from .bin
Select .bin files and create matching MODE2/2352 .cue files if missing. Useful for retro disc images.
Weekly Maintenance
Flatpak updates, journal vacuum, TRIM, temp cleanup, disk checks - the full weekly routine.
Reset Steam Client
Renames your Steam directory to Steam_backup. Steam will re-create a fresh directory on next launch. You'll need to sign in again.
Reset Gamescope
Clears Gamescope state and config directories. Fixes display and resolution glitches. Reboot recommended after.
Duplicate ROM Finder
Scans each console folder for near-duplicate ROM files using normalized filenames. Helps free up SD card space.
Find Lost ROMs
Searches for ROM files outside standard EmuDeck directories. Finds misplaced or forgotten ROMs across your Deck.
ROM Size Sorter
Lists all ROMs sorted by file size, largest first. Useful for identifying which games take up the most space.
Remove ROMs Metadata
Clears downloaded media and gamelist.xml files from EmuDeck and ES-DE. Useful for a clean scrape.
Large File Finder
Finds the largest files and directories on your Deck, sorted by size. Shows the top 50 results.
Deck Declutter
Removes known clutter files - old flatpaks, stale caches, leftover AppImages, and other junk.
Find Decky Leftovers
Searches for leftover Decky Loader files after uninstall. Shows sizes and locations of any remaining files.
Uninstall Decky Loader
Runs the official Decky uninstall script and cleans up all leftover system files, logs, and configs.
Full Recovery (Steam + Gamescope)
Nuclear option - resets both Steam client and Gamescope in one go. Use when stuck in a boot loop. Reboot after, then sign in to Steam again.
